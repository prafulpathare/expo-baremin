# expo boilerplate
boilarplate code for instant setup and development of expo based applications

### contains
1. default screens (home, profile, etc.)
2. bottom navigation
3. fonts
4. node dependancies
5. logos and icons

### setup
- install dependencies
- ```sh
  $ npm install
  ```
- run on local
  ```sh
  $ npm run start
  ```
- build
  ```sh
  $ npm run build:android
  $ npm run build:ios
  ```
